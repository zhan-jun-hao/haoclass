# 1.项目介绍

轻量版好课，能够实现：

## 1.1 客户端

1.1 用户登陆注册

1.2 能看到经典课程 能够切换课程分类查看

1.3 能够购买课 课也有优惠券 购买课时进行权限验证 可使用优惠券

1.4 能够调起微信支付买课，支持买一个系列的课

1.5 也可以支持全局VIP VIP有权限时间

1.6 进入视频后直接进入第1集的观看，支持切换集数，评论以及评论的评论，点赞，当前看到第几集

## 1.2 管理端

2.1 管理端是不同的后台管理网站同样需要登陆注册

2.2 统计管理：收益的查询

2.3 用户管理：用户的增删改查

2.4 权限管理：用户权限的增删改查

2.5 订单管理：订单的增删改查

2.6 营销管理：包含秒杀优惠券和发放优惠券的功能

2.7 课程管理：课程的增删改查

2.8 管理员操作日志

2.9 评论管理

## 1.3 服务端

3.1 提供客户端和管理端的数据支持

3.2 拥有网关 对用户权限进行控制

3.3 核心业务拆成微服务，非核心业务做成一个服务，采用DDD架构

# 2.架构设计

核心业务拆分成微服务，非核心业务采用DDD架构模式

```bash
gateway-service
main-service
pay-service
coupon-service
common-core
```
